== Description ==

This Plugin adds the required Author Meta Tags to the header of all the pages on your WordPress site.

*   You must have a Google+ Profile
*   You must add the domain you want to show your Author info beside to your "Contributer To" section of your Google+ Profile.

== Installation ==

This plugin was designed to be as simple as possible to use simply upload, activate, paste and done.

e.g.

1. Upload the plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Paste your Google+ Profile URL into the Google+ Author page in the settings menu

You must have add the domain to your "Contributer To" box on your Google+ profile.

== Frequently Asked Questions ==

= Does this work for multiple author sites =

No, this version of the plugin is specificlly designed for a super easy install for a single author site. A multi-author version will soon be available but it is slightly more complecated than this one to set up for the user. 

= Where do I paste my Google+ url? =

Navigate to Settings >> Google+ Author.
There you should be able to paste your url in the box provided.
